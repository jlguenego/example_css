Priority is concatenation of a + b + c + d
  1. a: 1 if style, 0 if selector
  2. b: number of # selector (id selector)
  3. c: number of class and attribute and pseudo-classes selector
  4. c: number of element and pseudo-element selector
  5. See this link.
  6. Example: body#foo>span#hello.bar[foo] 0222
