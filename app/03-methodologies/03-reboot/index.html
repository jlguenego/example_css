Reboot

Reboot is used by BootStrap to reset some CSS settings and incitate to use some rules.

  • box-sizing: border-box
  • Some element are rebooted. ie: table, p, etc. in order to be consistent between browsers and use "smarter" settings as for instance rem instead of em.
  • Avoid margin-top: this simplify the mental model about margin. vertically collapse, etc.
  • Prefer using rem for margin.
  • Minimize the font- properties
